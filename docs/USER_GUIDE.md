# CNshell 用户手册

## 首次连接

1. 点击工具栏 `+`，选择 SSH，填写名称、主机、端口和用户名。
2. 选择密码、私钥、SSH Certificate、SSH Agent 或 FIDO2 硬件密钥。私钥可用“选择…”打开 macOS 文件选择器，也可输入绝对路径；密码、私钥口令和只读安全 Bookmark 写入 macOS Keychain，私钥内容不会复制进 CNshell。FIDO2 只尝试 Agent 中的 `sk-*` 身份，普通 Agent 密钥不会被误用。
3. 首次连接时在服务器控制台或向管理员核对 SHA-256 主机指纹，再点击信任。
4. 指纹后续发生变化时 CNshell 会阻止连接；确认服务器确实重装或更换密钥后再更新记录。

## 终端与文件

- 终端标签互相独立；`⌘1…9` 切换标签，`⌘F` 搜索，`⌘K` 清屏，`⌘+` / `⌘-` / `⌘0` 调整或重置字号。
- “设置 → 终端”可调整字体、字号、行高、回滚行数、光标形状/闪烁和终端配色；保存后所有已打开终端即时生效，不会重建 SSH 会话。
- 连接编辑页可启用“覆盖全局终端偏好”，只让该连接使用独立外观和缓冲设置。
- 文件面板双击目录进入、双击小于 10 MB 的 UTF-8 文件编辑。
- 左侧目录树可逐级展开或折叠；点击目录名称会在右侧打开，并自动展开当前路径的祖先目录。
- 文件行右键可下载、上传到所选目录、新建文件/文件夹、打开、编辑、复制路径、重命名、改权限、压缩/解压和删除；新建文件不会覆盖已有同名文件。
- 保存文本会检查远端修改时间，先写临时文件再原子替换。
- 下载先写 `.cnshell-part`；成功同步后才改成目标文件名。
- 传输队列支持暂停、恢复、取消和失败重试。
- “上传文件夹”与选中目录后的“下载文件夹”会自动打包传输；覆盖时先完成新目录，再通过同级隐藏备份交换，失败或取消会清理临时归档并保留原目标。

## 监控

CNshell 每 2 秒通过独立 SSH Exec Channel 读取 `/proc`、`df`、`ip` 和 `ps`。CPU、网络上下行与延迟保留最近 5 分钟趋势，同时提供当前数值作为图表替代。监控失败不会中断终端与 SFTP，可在设置中调整刷新频率或暂停。

## 代理与隧道

在设置中创建 SOCKS5、HTTP CONNECT 或 SSH 跳板代理，再在连接编辑页选择。连接行的更多菜单可创建本地、远程和动态 SOCKS5 转发。非 loopback 监听可能暴露端口，必须自行配置防火墙。

连接可拖放到嵌套文件夹，也可从连接行的更多菜单选择“移动到文件夹”；删除的连接进入“已删除项目”，可恢复或永久删除。

主机密钥策略默认要求首次人工核对 SHA-256 指纹。可选的“自动信任首次密钥”仅适合可信网络：它会自动记录第一次看到的算法与指纹，之后任何变化仍会阻止连接。

## RDP

直接创建 RDP 连接即可。CNshell 应用包已内置 universal `sdl-freerdp`，使用动态分辨率、剪贴板和自动重连参数启动独立受管窗口，不需要 Homebrew 或 XQuartz。密码保存在 macOS Keychain，仅通过 sidecar stdin 传递，不进入命令行参数或环境变量。CNshell 标签会显示 sidecar 状态，关闭标签会终止对应远程桌面进程。

## 备份与诊断

- “安全导出”不含任何凭据。
- “加密导出凭据”要求至少 8 位口令，使用 Argon2id + AES-256-GCM；口令无法找回。
- 诊断 JSON 不包含主机、用户名、命令或文件路径，可在提交问题前自行检查。
- “设置 → 反馈与诊断”可打开预填版本、macOS 版本和架构的错误报告，也可提交功能建议或查看当前 Release。
- 诊断文件导出成功后可在 Finder 中显示；CNshell 不会自动上传诊断文件。

## 高级效率与协议

- 终端工具栏可打开智能命令栏、会话日志、高亮/通知、多主机批量执行、Copy Mode、跨标签搜索和逐行时间戳。
- 设置中的“高级协议与转发”会探测 Zmodem、SCP、Mosh、X Server 与 SSH Agent。Agent 转发必须逐连接启用，仅用于完全可信主机；设置只影响下一次连接。
- SFTP 子系统不可用时，普通文件传输会在同一条已认证、已校验指纹的 SSH 连接上自动尝试 SCP。SCP 上传只接受用户明确选择的覆盖策略。
- “受限任务编排”按顺序执行命令、匹配/条件和文件传输步骤。运行前必须检查最终预览；任一步失败即停止，可取消或修正后重试。
- “可选加密同步”只操作用户选择的本地目录，可指向 iCloud Drive、WebDAV 或 Git 的挂载/检出目录。口令默认不保存；可明确选择用 Touch ID 保存到这台 Mac 的设备专属 Keychain，之后生成或导入时由系统验证指纹。手动口令入口不会被移除，凭据同步默认关闭，冲突不会静默覆盖。
