# 快捷键

| macOS | Windows | 操作 |
| --- | --- | --- |
| `⌘N` | `Ctrl+N` | 新建连接 |
| `⌘T` | `Ctrl+T` | 打开新终端 |
| `⌘W` | `Ctrl+W` | 关闭当前会话；根据设置显示确认 |
| `⌘F` | `Ctrl+F` | 搜索当前终端输出 |
| `⌘⇧F` | `Ctrl+Shift+F` | 跨标签搜索终端输出 |
| `⌘K` | `Ctrl+K` | 清空终端可见缓冲 |
| `⌘+` | `Ctrl++` | 放大当前终端；按连接覆盖启用时只调整该连接 |
| `⌘-` | `Ctrl+-` | 缩小当前终端；按连接覆盖启用时只调整该连接 |
| `⌘0` | `Ctrl+0` | 将当前终端字号恢复为默认 13 px |
| `⌘J` | `Ctrl+J` | 展开/折叠会话工具区 |
| `⌘1…9` | `Ctrl+1…9` | 切换会话标签 |
| `⌘C` / `⌘V` | `Ctrl+C` / `Ctrl+V` | 有选区时复制；粘贴前按终端粘贴策略处理 |
| `⌘⇧C` | `Ctrl+Shift+C` | 进入 Copy Mode；方向键移动，`Enter` 复制，`Esc` 退出 |
| `⌘?` | `Ctrl+?` | 打开帮助 |
