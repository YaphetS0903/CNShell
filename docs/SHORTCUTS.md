# 快捷键

| 快捷键 | 操作 |
| --- | --- |
| `⌘N` | 新建连接 |
| `⌘T` | 打开新终端 |
| `⌘W` | 关闭当前会话；根据设置显示确认 |
| `⌘F` | 搜索终端输出 |
| `⌘K` | 清空终端可见缓冲 |
| `⌘+` | 放大当前终端；按连接覆盖启用时只调整该连接 |
| `⌘-` | 缩小当前终端；按连接覆盖启用时只调整该连接 |
| `⌘0` | 将当前终端字号恢复为默认 13 px |
| `⌘J` | 展开/折叠会话工具区 |
| `⌘1…9` | 切换会话标签 |
| `⌘⇧C` / `⌘⇧V` | 终端复制/粘贴（也兼容选中后 `⌘C`） |
| `⌘?` | 打开帮助 |
